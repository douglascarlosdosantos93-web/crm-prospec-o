1

Acesse notion.so/my-integrations → "New integration" → salve

2

Copie o Internal Integration Token (começa com ntn_)

3

No Notion, abra 📋 Lista de Prospecção → ⋯ → Connections → conecte

Fica salvo só no seu navegador. Nunca enviado a terceiros.
