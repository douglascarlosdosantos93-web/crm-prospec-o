🎯 CRM Prospecção

Cole o token da integração Notion para carregar os leads de Crespos & Cachos.

Como gerar o token (só 1x):

1

Acesse notion.so/my-integrations → "New integration" → salve

2

Copie o Internal Integration Token (começa com ntn_)

3

No Notion, abra 📋 Lista de Prospecção → ⋯ → Connections → conecte a integração

Fica salvo só no seu navegador. Nunca enviado a terceiros.
